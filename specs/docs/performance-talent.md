# Performance & Talent Management (การประเมินผลและการจัดการบุคลากร)

เอกสารนี้อธิบาย module ที่เกี่ยวกับการประเมินผลงาน การพัฒนาศักยภาพ และการวางแผนบุคลากร

---

## การประเมินผลงาน (`performance`)

**หน้าที่หลัก:** ระบบตั้งเป้าหมาย (Goal Setting) และติดตามความคืบหน้าผลงานประจำปี โดยใช้กรอบการประเมิน 3 ประเภทคือ KPI, G-BEST Competency และ Development Goal พร้อมระบบ Workflow การอนุมัติแบบ Sign-Off ระหว่างพนักงานและผู้จัดการ

**ใครใช้ได้บ้าง:** พนักงาน / ผู้จัดการ

**สิ่งที่ทำได้:**
- สร้างเป้าหมายใหม่ โดยกำหนด ชื่อเป้าหมาย (ภาษาไทย/อังกฤษ), หมวดหมู่ (KPI / G-BEST / Development), ประเภทการวัดผล (ตัวเลข / เปอร์เซ็นต์ / ใช่-ไม่ใช่ / คะแนน), ค่าเป้าหมาย, น้ำหนักคะแนน (%), ช่วงเวลาประเมิน และวันเริ่ม-สิ้นสุด
- กรองเป้าหมายตามปี (2024, 2025, 2026, รายไตรมาส), หมวดหมู่ และสถานะ
- ส่งเป้าหมายให้ผู้จัดการอนุมัติ (Submit for Review)
- แก้ไขเป้าหมายที่ยังเป็น Draft หรือถูกส่งกลับ (Sent Back)
- ลบเป้าหมายที่ยังเป็น Draft
- อัปเดตความคืบหน้า (Progress Update) พร้อมใส่ความคิดเห็น
- Sign-Off เป้าหมายที่ได้รับอนุมัติแล้ว (ทั้งฝั่งพนักงานและผู้จัดการ)
- ดูรายละเอียดเป้าหมาย รวมถึงสถานะ Sign-Off และ comment ของผู้อนุมัติ
- ดูแผนภาพวงกลม (Progress Ring) ภาพรวมความสำเร็จโดยรวม
- ดูประวัติการเปลี่ยนแปลงเป้าหมาย เช่น การสร้าง, ส่ง, อนุมัติ, ส่งกลับ

**ข้อมูลที่แสดง:**
- สรุปน้ำหนักคะแนนรวม (ต้องเท่ากับ 100%) แยกตามหมวด KPI / G-BEST / Development
- รายการเป้าหมายพร้อม Progress Bar, สถานะ, น้ำหนัก, ช่วงเวลา
- แท็บ "เป้าหมาย" — รายการเป้าหมายทั้งหมดพร้อม Filter
- แท็บ "ความคืบหน้า" — ภาพรวมอัตราความสำเร็จ (Weighted Progress)
- แท็บ "การประเมิน" — แบบฟอร์มประเมินผลตนเองและดูคะแนนของผู้จัดการ
- แท็บ "ประวัติ" — Timeline การเปลี่ยนแปลงทั้งหมดย้อนหลัง 20 รายการ

**หมายเหตุ:**
- เป้าหมายที่ถูกส่งกลับ (Sent Back) จะให้แก้ไขและส่งใหม่ได้
- น้ำหนักรวมทุกเป้าหมายต้องเท่ากับ 100% ก่อนจึงจะถือว่า Valid
- G-BEST Competency Code จะปรากฏให้เลือกเฉพาะเมื่อเลือกหมวดหมู่ G-BEST
- Workflow Sign-Off ต้องทำทั้งสองฝั่ง (พนักงาน + ผู้จัดการ) จึงจะปิดวงจรสมบูรณ์

## Scorecard และ KPI (`scorecard`)

**หน้าที่หลัก:** หน้าแสดงผล Scorecard ส่วนบุคคล ที่รวบรวมข้อมูลการประเมินสมรรถนะ (Competency), ผลการประเมิน, จุดแข็ง, ด้านที่ต้องพัฒนา, เป้าหมายอาชีพ และตำแหน่งในแมทริกซ์ 9 ช่อง (9-Box Grid) ไว้ในที่เดียว

**ใครใช้ได้บ้าง:** พนักงาน / ผู้จัดการ / HR Admin / HR Manager

**สิ่งที่ทำได้:**
- ดู CG Competency Assessment — รายการสมรรถนะองค์กรพร้อมคะแนน (1-5 ดาว)
- ดูประวัติการประเมิน (Assessment History) — รายการโปรแกรมประเมินที่ผ่านมา พร้อมผู้ประเมินและวันที่
- ดูสรุปผลการประเมิน (Assessment Summary) — คะแนนรวม Overall Rating / 5.0 และรอบที่ประเมิน
- ดูความสำเร็จสำคัญ (Key Successes) — ผลงานเด่นพร้อมระดับผลกระทบ (High / Medium / Low)
- ดูจุดแข็งและด้านที่ต้องพัฒนา (Top Strengths & Development Areas) แบบเปรียบเทียบ Side-by-Side
- ดูเป้าหมายอาชีพระยะสั้นและระยะยาว (Career Aspirations) พร้อม Timeframe
- ดูวัตถุประสงค์การพัฒนา (Development Objectives) พร้อม Progress Bar และวันสิ้นสุด
- ดู Talent Reference — ข้อมูลกลุ่ม Talent Pool, คะแนน Potential Rating, ความพร้อมในการเลื่อนขั้น และตำแหน่งที่วางแผนสืบทอด
- ดูตำแหน่งใน 9-Box Grid (Performance-Potential Matrix) — แสดงตำแหน่งของพนักงานใน 9 ช่อง เช่น Stars, Future Leader, Core
- ดูผล Final Calibration — คะแนนสุดท้ายหลัง Calibration Session พร้อมชื่อผู้ Calibrate และความคิดเห็น

**ข้อมูลที่แสดง:**
- CG Competency พร้อม Rating Badge (1-5) และคำอธิบาย
- 9-Box Grid แสดงตำแหน่งของพนักงานด้วย Icon และ Highlight
- ป้ายระดับผลกระทบ (Impact Badge): High (แดง), Medium (เหลือง), Low (น้ำเงิน)
- คะแนน Overall Final Calibration จากผู้ Calibrate พร้อมวันที่และ Comment

**หมายเหตุ:**
- หน้า Scorecard แสดงจากข้อมูลของพนักงานคนปัจจุบัน (Employee Profile)
- ทุก Section สามารถย่อ/ขยายได้ (Collapsible Card)
- 9-Box Grid มี 9 ช่อง ตั้งแต่ Questionable (ต่ำ-ต่ำ) ถึง Stars (สูง-สูง)

## การจัดการความสามารถพิเศษ (`talent-management`)

**หน้าที่หลัก:** ระบบระบุและจัดการพนักงาน High Potential (Hi-Po) ขององค์กร ผ่านการแสดงผล 9-Box Grid แบบโต้ตอบได้ พร้อม Dashboard ภาพรวมการกระจายตัวของ Talent และ Calibration Review Session

**ใครใช้ได้บ้าง:** ผู้จัดการ / HR Admin / HR Manager

**สิ่งที่ทำได้:**
- ดู Dashboard ภาพรวมจำนวน Hi-Po, Stars, At-Risk และพร้อมเลื่อนขั้น (Ready Now) แยกตามแผนก
- ดู 9-Box Grid แบบ Interactive — ดูรายชื่อพนักงานในแต่ละช่องพร้อมย้ายพนักงานโดย Drag-and-Drop
- กรองข้อมูลใน 9-Box Grid ตามแผนก
- ดูรายชื่อ Hi-Po Employees พร้อม Performance Rating, Potential Rating และความเสี่ยงที่จะลาออก (Risk of Loss)
- ค้นหาและกรอง Talent Pool ตามแผนก, ความเสี่ยงการสูญเสีย (High / Medium / Low)
- ดูโปรไฟล์ Talent รายบุคคล (Talent Profile Modal) รวมถึง คะแนนผลงาน, อายุงาน, ตำแหน่งใน 9-Box, ความพร้อมเลื่อนขั้น
- ดูและกำหนดเกณฑ์ Hi-Po (Criteria Configuration) — เกณฑ์คะแนนผลงาน, Potential, อายุงานขั้นต่ำ และอายุสูงสุด
- ดูประวัติ Calibration Review Session — วันที่จัดประชุม, จำนวนพนักงานที่ผ่านการ Review, Action Items
- เริ่ม Calibration Session ใหม่

**ข้อมูลที่แสดง:**
- Summary Cards: จำนวน Hi-Po ทั้งหมด, Stars, At-Risk, Ready Now พร้อมสัดส่วนต่อ Workforce
- ตาราง Distribution ตามแผนก: รวม / Hi-Po / Stars / At-Risk
- 9-Box Grid แต่ละช่องแสดงจำนวนพนักงานและชื่อ
- Talent Card แต่ละคน: ชื่อ, ตำแหน่ง, แผนก, Performance Rating, Potential Rating, อายุงาน, ป้าย Hi-Po/Stars/At-Risk

**หมายเหตุ:**
- เกณฑ์ Hi-Po (Criteria) แก้ไขได้เฉพาะผู้มีสิทธิ์ `edit_talent_criteria` เท่านั้น
- Drag-and-Drop ใน 9-Box Grid ใช้ย้ายพนักงานระหว่างช่องเพื่อ Calibrate ผล
- พนักงานที่มี Risk of Loss สูงจะแสดง icon เตือน (warning) ใน 9-Box Grid

## การวางแผนสืบทอดตำแหน่ง (`succession-planning`)

**หน้าที่หลัก:** ระบบวางแผนการสืบทอดตำแหน่งสำคัญขององค์กร ตั้งแต่การระบุตำแหน่งที่มีความเสี่ยง การเสนอชื่อผู้สืบทอด การแสดง Succession Chart และการติดตาม Development Actions ของผู้ที่ถูกเสนอชื่อ

**ใครใช้ได้บ้าง:** ผู้จัดการ / HR Admin / HR Manager (พนักงานทั่วไปไม่มีสิทธิ์เข้าถึง)

**สิ่งที่ทำได้:**
- ดู Dashboard ภาพรวม: จำนวนตำแหน่งสำคัญ, อัตราความครอบคลุม (Coverage Ratio), อัตราพร้อม (Ready Now), ตำแหน่งเสี่ยงสูง, Action Items ที่ครบกำหนด
- ดู Risk Heat Map แสดงตำแหน่งแยกตามระดับความเสี่ยง (High / Medium / Low) พร้อมเหตุผล
- เสนอชื่อผู้สืบทอด (Nominate Successor)
- ดูและกรองรายการตำแหน่งสำคัญ ตามระดับความเสี่ยงและสถานะ Coverage
- ค้นหาตำแหน่งตามชื่อ, แผนก หรือชื่อผู้ดำรงตำแหน่งปัจจุบัน
- ดูรายละเอียดตำแหน่ง: ผู้ดำรงตำแหน่งปัจจุบัน, อายุในตำแหน่ง, วันเกษียณ (ถ้ามี), จำนวนผู้สืบทอด, จำนวนที่ "Ready Now"
- ดูรายชื่อผู้สืบทอดทั้งหมดพร้อมระดับความพร้อม
- ดู Succession Chart แบบ Org-Chart Tree แสดงสายการสืบทอด
- ดูและติดตาม Development Actions รายบุคคลของผู้สืบทอด พร้อมสถานะและวันครบกำหนด

**ข้อมูลที่แสดง:**
- Metric Cards: จำนวนตำแหน่งสำคัญ, Coverage Ratio (%), Ready Now Ratio (%), High-Risk Positions, Actions Due
- ตาราง Coverage Summary: ตำแหน่ง, ผู้ดำรงตำแหน่ง, ระดับเสี่ยง, สถานะครอบคลุม, จำนวนผู้สืบทอด
- Position Card: สีตามระดับเสี่ยง (แดง/เหลือง/เขียว), ระยะเวลาในตำแหน่ง, วันเกษียณ

**หมายเหตุ:**
- หน้านี้เข้าถึงได้เฉพาะ Manager ขึ้นไปและ HR เท่านั้น พนักงานทั่วไปจะเห็นหน้า "ไม่มีสิทธิ์เข้าถึง"
- ตำแหน่งสำคัญสามารถกรองได้ 3 ระดับ: Coverage (Full / Partial / At Risk) และ Risk (High / Medium / Low)
- Development Actions ที่ค้างอยู่จะแสดงใน Dashboard Widget "Upcoming Actions" พร้อมผู้รับผิดชอบ

## แผนพัฒนารายบุคคล (`idp`)

**หน้าที่หลัก:** ระบบวางแผนการพัฒนาศักยภาพรายบุคคล (Individual Development Plan) โดยเริ่มจากการวิเคราะห์ช่องว่างสมรรถนะ (Competency Gap Analysis) ออกแบบแผนงาน กำหนด Development Actions และติดตามความก้าวหน้า จนถึงการจัดเก็บ Portfolio ผลงาน

**ใครใช้ได้บ้าง:** พนักงาน / ผู้จัดการ

**สิ่งที่ทำได้:**
- ดูและวิเคราะห์ Competency Gap Analysis — เปรียบเทียบระดับสมรรถนะปัจจุบันกับที่ต้องการ แสดงเป็น Progress Bar และระดับ Gap
- กรอง Gap Analysis ตามระดับ Priority (High / Medium / Low)
- เพิ่ม Development Action จากช่องว่างสมรรถนะที่ต้องการพัฒนา
- เลือกประเภท Development Action: Training, Coaching, Mentoring, On-the-Job หรือประเภทอื่น
- สร้าง IDP ใหม่ (New IDP) สำหรับปีที่ต้องการ
- ส่ง IDP ขออนุมัติ (Submit for Approval) ในสถานะ Draft
- ดูรายละเอียด IDP: สถานะ, วันสร้าง, ความก้าวหน้ารวม, ผู้อนุมัติ, Sign-Off ของพนักงานและผู้จัดการ
- ดูรายการ Development Actions พร้อม Progress Bar, Milestones, ลิงก์ LMS
- ดูความก้าวหน้าแยกตาม Development Area (แต่ละสมรรถนะที่พัฒนา)
- ขอ Competency Reassessment
- ดู Portfolio — ประวัติ IDP ทุกปี และ Certifications ที่ได้รับ

**ข้อมูลที่แสดง:**
- Summary Cards ใน Gap Analysis: จำนวนสมรรถนะ, High Priority Gaps, Total Gap Levels, Average Gap
- Progress Tracker รายสมรรถนะ: ระดับปัจจุบัน → ระดับที่ต้องการ แสดงด้วย 5 ช่องสี
- Action Card: ประเภท, ชื่อ, รายละเอียด, ช่วงวันที่, Milestones (Checklist), Progress %, สถานะ
- IDP Form: สถานะ, วันสร้าง, ผู้อนุมัติ, Overall Progress, Sign-Off Status

**หมายเหตุ:**
- IDP แต่ละปีมีสถานะ: Draft → Submitted → Approved / Rejected → Cancelled
- เมื่อ IDP ได้รับอนุมัติ ต้องมี Sign-Off ทั้งพนักงานและผู้จัดการ
- แต่ละ Development Action สามารถมี Milestones ย่อยและลิงก์ไปยังระบบ LMS ได้

## การเรียนรู้และฝึกอบรม (`learning`)

**หน้าที่หลัก:** ระบบ Learning & Development ที่รวมหลักสูตรฝึกอบรมทั้งหมดไว้ใน Catalog พนักงานสามารถค้นหา สมัครเรียน ดูตารางอบรม และติดตาม My Learning Journey รวมถึงปฏิทินการจัดอบรม

**ใครใช้ได้บ้าง:** พนักงาน / ผู้จัดการ / HR Admin / HR Manager

**สิ่งที่ทำได้:**
- ค้นหาหลักสูตรใน Catalog ตามชื่อ, รหัสวิชา, หมวดหมู่, ระดับ (Beginner / Intermediate / Advanced) และรูปแบบการเรียน
- กรองหลักสูตรตาม Delivery Method: Classroom, Online, Blended, On-the-Job Training
- ดูรายละเอียดหลักสูตร: ชื่อ, รหัส, ระยะเวลา (ชั่วโมง), Credits, รูปแบบการเรียน, จำนวนผู้เรียนสูงสุด, วัตถุประสงค์การเรียนรู้, Prerequisites, รายชื่ออาจารย์
- สมัครเรียน (Enroll) ผ่านระบบ
- ดูตาราง Class ที่เปิดสอนของแต่ละหลักสูตร
- ดู My Learning — รายการหลักสูตรที่ลงทะเบียนไว้พร้อมสถานะ (Enrolled / In Progress / Completed)
- ดู Roadmap — แนะนำเส้นทางการเรียนรู้ที่เหมาะสมตามตำแหน่งหรือสมรรถนะ
- ดูปฏิทิน (Calendar) — ตารางการจัดอบรมรายเดือน

**ข้อมูลที่แสดง:**
- Quick Stats: หลักสูตรที่เสร็จแล้ว, กำลังเรียน, Credits ที่ได้รับ, จำนวน Certificates
- Course Card: หมวดหมู่ (พร้อมสี), ชื่อ, รหัส, ระยะเวลา, รูปแบบการเรียน, ระดับ, Rating ดาว (จาก Review), สถานะ Enrollment, จำนวน Class ที่เปิด
- ป้าย Mandatory แสดงหลักสูตรบังคับ
- Prerequisites พร้อมสีบอกว่าเรียนผ่านแล้วหรือยัง (เขียว = ผ่าน, เหลือง = ยังไม่ผ่าน)

**หมายเหตุ:**
- หลักสูตรที่มี Prerequisites ที่ยังไม่ผ่านจะแสดง Warning ในหน้ารายละเอียด
- ป้าย "Mandatory" แสดงในกรณีที่หลักสูตรเป็น Mandatory Training สำหรับพนักงาน
- สถานะ Enrollment ของแต่ละหลักสูตรแสดงบน Course Card ทันที (เช่น กำลังเรียน / เสร็จแล้ว)

## บันทึกการฝึกอบรม (`training-records`)

**หน้าที่หลัก:** ระบบบันทึกและรายงานประวัติการฝึกอบรมรายบุคคลแบบครบวงจร ตั้งแต่บันทึกผลการอบรม การประเมินผลตามโมเดล Kirkpatrick 4 ระดับ การออกและดาวน์โหลด Certificate ไปจนถึงรายงานสถิติและการเชื่อมโยงกับ Competency

**ใครใช้ได้บ้าง:** พนักงาน / ผู้จัดการ / HR Admin / HR Manager

**สิ่งที่ทำได้:**
- ดูและกรองประวัติการฝึกอบรมทั้งหมด ตามปี, หมวดหมู่ (Leadership / Technical / Soft Skills / Compliance / Safety / Professional) และสถานะ
- ดูรายละเอียดแต่ละ Training Record: ชื่อหลักสูตร, รหัส, ผู้จัด (Provider), ระยะเวลา, ช่วงวันที่, สถานที่, ประเภท (Mandatory / Voluntary)
- ดูคะแนน Pre-Assessment และ Post-Assessment พร้อมอัตราการพัฒนา (Improvement)
- ดาวน์โหลด Certificate จากหน้า Training Record Card โดยตรง
- ดูรายการประเมินผลตามโมเดล Kirkpatrick: Reaction, Learning, Behavior, Results และ Instructor Rating
- ดูรายละเอียดการประเมินแต่ละหลักสูตร
- ดู Certificate Gallery — บัตรแสดง Certificate ทั้งหมด พร้อมผู้ออก, วันออก, วันหมดอายุ
- ดาวน์โหลด Certificate และตรวจสอบความถูกต้องผ่าน QR Code
- รับการแจ้งเตือนเมื่อ Certificate ใกล้หมดอายุ (ภายใน 90 วัน)
- ดูรายงาน Training Transcript — ชั่วโมงอบรมแยกตามหมวดหมู่และแยกตามปี พร้อมค่าใช้จ่าย
- ดาวน์โหลด Transcript
- ดูรายงาน Department Summary: อัตราการเข้าร่วม, ชั่วโมงเฉลี่ย/คน, Budget Utilization, อัตรา Mandatory Completion, Budget vs Actual
- ดู Top Courses ของแผนก
- ดูความเชื่อมโยงระหว่างการฝึกอบรมกับ Competency — ระดับสมรรถนะปัจจุบัน/เป้าหมาย, รายการ Training ที่เชื่อมโยง
- ดูความก้าวหน้าของ IDP และ Skill Matrix

**ข้อมูลที่แสดง:**
- Summary Cards: ชั่วโมงอบรมรวม, หลักสูตรที่เสร็จแล้ว, กำลังดำเนินการ, จำนวน Certificates
- Training Record Card: หมวดหมู่, สถานะ (เสร็จ/กำลังเรียน/สมัครแล้ว/แผน/ยกเลิก/ไม่ผ่าน), Mandatory flag, คะแนน Pre/Post Assessment, ปุ่มดาวน์โหลด Certificate
- Evaluation Table (Kirkpatrick): คะแนน Reaction / Learning / Behavior / Results / Instructor Rating พร้อม Badge สี
- Certificate Card: ชื่อหลักสูตร, หมายเลข Certificate, ผู้ออก, วันออก, วันหมดอายุ (พร้อมแจ้งเตือน)

**หมายเหตุ:**
- Certificate ที่ใกล้หมดอายุ (ภายใน 90 วัน) จะแสดง Warning Banner ด้านบนของ Certificate tab
- Certificate ที่หมดอายุแล้วจะแสดงสถานะ "Expired" สีแดง
- การตรวจสอบ Certificate ทำได้ผ่านปุ่ม QR Code ในแต่ละ Certificate Card
- รายงาน Kirkpatrick ใช้ 4 ระดับ: Level 1 Reaction, Level 2 Learning, Level 3 Behavior, Level 4 Results

## ระบบสิทธิ์การเข้าถึง (RBAC) สำหรับ Module ทั้ง 7

ระบบใช้ Role-Based Access Control (RBAC) โดยมี 4 Role และการสืบทอดสิทธิ์ดังนี้:

| Role | สืบทอดสิทธิ์จาก |
|------|----------------|
| `employee` | — |
| `manager` | employee |
| `hr_admin` | manager + employee |
| `hr_manager` | hr_admin + manager + employee |

**สรุปสิทธิ์ที่เกี่ยวข้องกับ Module นี้:**

| Module | Employee | Manager | HR Admin | HR Manager |
|--------|----------|---------|----------|------------|
| performance | ตั้งเป้าหมาย, อัปเดต, Sign-Off | ดูทีม, อนุมัติ Goal | ดูทั้งหมด | ดูทั้งหมด |
| scorecard | ดูของตัวเอง | ดูทีม | ดูทั้งหมด | ดูทั้งหมด |
| talent-management | ไม่มีสิทธิ์ | ดูทีม | จัดการ Talent Pool | กำหนดเกณฑ์ |
| succession-planning | ไม่มีสิทธิ์ | ดูและ Nominate | จัดการ | อนุมัติ Sensitive |
| idp | สร้างและส่ง IDP | ดูทีม, อนุมัติ | ดูทั้งหมด | ดูทั้งหมด |
| learning | ดูและสมัครเรียน | ดูทีม | ดูและจัดการ | ดูทั้งหมด |
| training-records | ดูของตัวเอง | ดูทีม | ดูรายงาน | ดูรายงานทั้งหมด |

**Workflow ที่ต้องการการอนุมัติพิเศษ (Sensitive Workflows):**
- `compensation_change` — ต้องการ HR Manager
- `termination` — ต้องการ HR Manager
- `promotion` — ต้องการ HR Manager
